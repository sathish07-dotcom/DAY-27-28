DOM (Document Object Model)

The **DOM (Document Object Model)** in JavaScript is a programming interface for web documents.
It represents the structure of a web page as a tree of nodes, where each node is an object corresponding to a part of the page.
The DOM allows JavaScript to interact with and manipulate the page’s structure, content, and styles dynamically.

Key Concepts of DOM:

Document Object Model (DOM)
  • The DOM is essentially a hierarchical tree structure that represents an HTML or XML document.
  • Every element, attribute, and piece of text in an HTML document is represented as a node in this tree.
  • The DOM provides an interface for manipulating these nodes.

    • Will be represented in the DOM like this:
    Document
    └─ HTML
    └─ BODY
    └─ DIV (id="container")
    └─ P
    └─ TextNode ("Hello, world!")

    DOM Tree
  • The DOM structure is often visualized as a tree, where:
  • he **root** of the tree is the `document` object.
  • **Branches** are elements like `html`, `body`, `div`, etc.
  • **Leaves** are text nodes (the content within the tags).

    • Core Operations in the DOM:

    Selecting Elements:

  • You need to select elements to manipulate them using JavaScript.

    • `document.getElementById(id)`**: Selects an element by its ID.

    `document.getElementsByClassName(className)`**: Selects elements by their class name.

    `document.getElementsByTagName(tagName)`**: Selects elements by their tag name.

    `document.querySelector(selector)`**: Selects the first element matching a CSS selector.

    `document.querySelectorAll(selector)`**: Selects all elements matching a CSS selector.

    let element = document.getElementById("myElement"); // Select by ID let allDivs = document.getElementsByTagName("div"); // Select all div elements let firstButton = document.querySelector("button"); // Select first button element

    Modifying Content:

  • You can change the content inside selected elements.

    • `innerHTML`**: Changes the HTML content inside an element.

    `textContent`**: Changes the text content of an element.

    element.innerHTML = "New HTML Content"; // Replace inner HTML element.textContent = "New Text Content"; // Replace text content

    Modifying Attributes:

  • You can modify the attributes of elements.

    • `getAttribute(attributeName)`**: Retrieves the value of an attribute.

    `setAttribute(attributeName, value)`**: Sets the value of an attribute.

    `removeAttribute(attributeName)`**: Removes an attribute.

    let srcValue = element.getAttribute("src"); // Get the 'src' attribute element.setAttribute("src", "new-image.jpg"); // Change the 'src' attribute element.removeAttribute("class"); // Remove the 'class' attribute

    >Modifying Styles:

  • You can manipulate the CSS styles of elements.

    • `style`**: Modifies the inline styles of an element.

    element.style.color = "red"; // Set the text color to red element.style.fontSize = "20px"; // Change font size to 20px

    Creating and Adding New Elements:

  • You can create new elements and add them to the DOM.
    • - **`document.createElement(tagName)`**: Creates a new element. - **`element.appendChild(child)`**: Adds a child element to an existing element. - **`element.insertBefore(newElement, referenceElement)`**: Inserts a new element before a reference element. ```javascript let newDiv = document.createElement("div"); newDiv.textContent = "I am a new div!"; document.body.appendChild(newDiv); // Add the new div to the body ``` #### 6. **Removing Elements**: You can remove elements from the DOM. - **`parentElement.removeChild(child)`**: Removes a child element from the parent. - **`element.remove()`**: Removes the element from the DOM. ```javascript let element = document.getElementById("removeMe"); element.remove(); // Removes the element from the DOM ``` #### 7. **Event Handling**: JavaScript can respond to user interactions by attaching event listeners to elements. - **`addEventListener(event, function)`**: Attaches an event listener to an element. - **`removeEventListener(event, function)`**: Removes an event listener from an element. ```javascript let button = document.getElementById("myButton"); button.addEventListener("click", function() { alert("Button clicked!"); }); button.removeEventListener("click", function() { alert("Button clicked!"); }); ``` #### 8. **Traversing the DOM**: You can navigate through the DOM tree to access different elements. - **`parentNode`**: Access the parent of an element. - **`childNodes`**: Access all child nodes (including text nodes). - **`firstChild` / `lastChild`**: Access the first or last child node. - **`nextSibling` / `previousSibling`**: Access the next or previous sibling element. ```javascript let div = document.getElementById("container"); let parent = div.parentNode; // Get parent element let firstChild = div.firstChild; // Get first child element let nextSibling = div.nextSibling; // Get next sibling element ``` ### Example: DOM Manipulation in Action Here’s a complete example where we: 1.
